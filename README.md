# 🎰 로또 미션

로또 발매기 프로그램을 구현합니다.

<br>

## 📋 기능 목록

### 💰 로또 구매
- [x] 로또 구매 금액 입력 받기
- [x] 구매 금액 검증 (1,000원 단위)
- [x] 구매한 로또 수량 계산
- [x] 로또 번호 자동 생성 (1~45 중 중복되지 않는 6개)
- [x] 발행한 로또 출력

### 🎯 당첨 번호 입력
- [x] 당첨 번호 6개 입력 받기 (쉼표로 구분)
- [x] 보너스 번호 입력 받기
- [x] 당첨 번호 검증 (범위, 중복, 개수)
- [x] 보너스 번호 검증 (범위, 당첨 번호와 중복)

### 📊 당첨 결과 계산
- [x] 구매한 로또와 당첨 번호 비교
- [x] 일치하는 번호 개수 계산
- [x] 보너스 번호 일치 여부 확인
- [x] 당첨 등급 판정 (1~5등)

### 📈 결과 출력
- [x] 당첨 내역 출력
- [x] 총 수익률 계산 및 출력 (소수점 둘째 자리 반올림)

### ⚠️ 예외 처리
- [x] 구매 금액이 1,000원 단위가 아닌 경우
- [x] 로또 번호가 1~45 범위를 벗어나는 경우
- [x] 로또 번호가 중복되는 경우
- [x] 로또 번호 개수가 6개가 아닌 경우
- [x] 보너스 번호가 당첨 번호와 중복되는 경우
- [x] 모든 예외는 IllegalArgumentException 발생 및 "[ERROR]"로 시작하는 메시지 출력

<br>

## ✅ 요구사항 체크리스트

### 🎯 기능 요구사항
- [x] 로또 번호 범위 1~45: `Constants.MIN_NUMBER`, `Constants.MAX_NUMBER` 사용
- [x] 중복 없는 6개 숫자: `Lotto` 생성자에서 검증
- [x] 당첨 번호 6개 + 보너스 번호 1개: `InputHandler`에서 입력 처리
- [x] 당첨 등급 1~5등: `LottoRank` Enum 구현
- [x] 로또 1장 = 1,000원: `Constants.LOTTO_PRICE` 사용
- [x] 예외 처리: `IllegalArgumentException`, `[ERROR]` 메시지
- [x] 입력/출력 형식: 요구 형식 준수

### 💻 프로그래밍 요구사항
- [x] JDK 21: `build.gradle`에 설정
- [x] Application.main(): 진입점 설정
- [x] System.exit() 미사용: 코드에 없음
- [x] Scanner 미사용: `Console.readLine()` 사용
- [x] Indent depth 3 이하: 모든 코드 depth 2 이하
- [x] 메서드 15라인 이하: 모든 메서드 확인 완료
- [x] else 미사용: `if` + `return` 패턴 사용
- [x] switch/case 미사용: 코드에 없음
- [x] 3항 연산자 미사용: 코드에 없음
- [x] Enum 사용: `LottoRank` 구현
- [x] 상수 사용: `Constants` 클래스로 분리
- [x] 하드코딩 제거: 모든 값 상수화

### 📦 Lotto 클래스 제약사항
- [x] `numbers` 필드 `private final` 유지
- [x] `numbers` 외 필드 추가 없음
- [x] 패키지 변경: `lotto.domain`으로 이동 (허용됨)

### 📝 입출력 요구사항
- [x] 출력 형식: 요구 형식 준수
- [x] 로또 번호 오름차순 정렬: `generateLotto()`에서 정렬
- [x] 수익률 소수점 둘째 자리 반올림: `Math.round()` 사용

```
구입금액을 입력해 주세요.
8000

[8, 21, 23, 41, 42, 43] 
[3, 5, 11, 16, 32, 38] 
[7, 11, 16, 35, 36, 44] 
[1, 8, 11, 31, 41, 42] 
[13, 14, 16, 38, 42, 45] 
[7, 11, 30, 40, 42, 43] 
[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]

당첨 번호를 입력해 주세요.
1,2,3,4,5,6

보너스 번호를 입력해 주세요.
7

당첨 통계
---
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
총 수익률은 62.5%입니다.
```
