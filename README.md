# 🎰 로또 미션

로또 발매기 프로그램을 구현합니다.

<br>

## 📋 기능 목록

### 💰 로또 구매
- [x] 로또 구매 금액 입력 받기
- [x] 구매 금액 검증 (1,000원 단위)
- [x] 구매한 로또 수량 계산
- [x] 로또 번호 자동 생성
  * 1~45 중 중복되지 않는 6개 번호 랜덤 생성
  * `Randoms.pickUniqueNumbersInRange()` 사용
  * 생성된 번호 오름차순 정렬
  * 구매 금액에 맞는 개수만큼 로또 생성
- [x] 로또 번호 검증
  * 번호 개수 검증 (6개 여부)
  * 번호 범위 검증 (1~45 범위)
  * 번호 중복 검증
- [x] 발행한 로또 출력

### 🎯 당첨 번호 입력
- [x] 당첨 번호 6개 입력 받기 (쉼표로 구분)
- [x] 보너스 번호 입력 받기
- [x] 당첨 번호 검증
  * 번호 범위 검증 (1~45 범위)
  * 번호 중복 검증
  * 번호 개수 검증 (6개 여부)
- [x] 보너스 번호 검증
  * 번호 범위 검증 (1~45 범위)
  * 당첨 번호와 중복 검증

### 📊 당첨 결과 계산
- [x] 구매한 로또와 당첨 번호 비교
  * 각 로또의 번호를 당첨 번호와 비교
  * 일치하는 번호의 개수 계산
- [x] 보너스 번호 일치 여부 확인
  * 각 로또에 보너스 번호가 포함되어 있는지 확인
- [x] 당첨 등급 판정
  * 일치 번호 개수와 보너스 볼 여부로 등급 결정
  * 1등: 6개 일치 (20억원)
  * 2등: 5개 일치 + 보너스 볼 (3천만원)
  * 3등: 5개 일치 (150만원)
  * 4등: 4개 일치 (5만원)
  * 5등: 3개 일치 (5천원)
  * 당첨 없음: 2개 이하 일치
- [x] 등급별 당첨 개수 집계
  * 각 등급별로 당첨된 로또 개수를 집계
- [x] 총 당첨금 계산
  * 각 등급의 상금 × 해당 등급 당첨 개수를 합산
- [x] 수익률 계산 및 출력
  * 수익률 = (총 당첨금 / 구매 금액) × 100
  * 소수점 둘째 자리에서 반올림하여 출력

### 📈 결과 출력
- [x] 당첨 내역 출력
- [x] 총 수익률 출력

### ⚠️ 예외 처리
- [x] 구매 금액이 숫자가 아닌 경우
- [x] 구매 금액이 1,000원 단위가 아닌 경우
- [x] 구매 금액이 최소 1,000원 미만인 경우
- [x] 로또 번호가 숫자가 아닌 경우
- [x] 로또 번호가 1~45 범위를 벗어나는 경우
- [x] 로또 번호가 중복되는 경우
- [x] 로또 번호 개수가 6개가 아닌 경우
- [x] 보너스 번호가 숫자가 아닌 경우
- [x] 보너스 번호가 1~45 범위를 벗어나는 경우
- [x] 보너스 번호가 당첨 번호와 중복되는 경우
- [x] 모든 예외는 `IllegalArgumentException` 또는 `NumberFormatException` 발생 및 "[ERROR]"로 시작하는 메시지 출력
- [x] 예외 발생 시 에러 메시지를 출력하고 해당 입력을 다시 받음

<br>

## 🎮 실행 결과 예시

```
구입금액을 입력해 주세요.
8000

8개를 구매했습니다.
[8, 21, 23, 41, 42, 43] 
[3, 5, 11, 16, 32, 38] 
[7, 11, 16, 35, 36, 44] 
[1, 8, 11, 31, 41, 42] 
[13, 14, 16, 38, 42, 45] 
[7, 11, 30, 40, 42, 43] 
[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]

당첨 번호를 입력해 주세요.
1,2,3,4,5,6

보너스 번호를 입력해 주세요.
7

당첨 통계
---
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
총 수익률은 62.5%입니다.
```
